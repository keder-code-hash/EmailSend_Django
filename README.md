# EmailSend_Django
## Details :
    A simple APi for sending mail to any one.
## web API url : 
<pre>
<code>
https://sending1email.herokuapp.com/send_email
</code>
</pre>
**only POST request is supported**
## Request Body Format : 
<pre>
<code>
{  
    "username" : "XXXX@hash",
    "send_to" : "XXXXXX@gmail.com"
}
</code>
</pre>
